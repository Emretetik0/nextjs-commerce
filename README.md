**Next.js Commerce**, modern e-ticaret siteleri için geliştirilmiş, yüksek performanslı ve tam ölçeklenebilir bir storefront çözümüdür. Hız, SEO ve kullanıcı deneyimi odaklı mimarisiyle en zorlu e-ticaret senaryolarını karşılar.

## ✨ Temel Özellikler

- **⚡ Yıldırım Hızında:** Next.js App Router ve React Server Components ile optimize edilmiş yükleme süreleri.
- **🛍️ Headless Mimari:** Shopify Storefront API ile kusursuz entegrasyon (Mock data katmanı ile demo modu aktif).
- **🎨 Modern Tasarım:** Tailwind CSS 4 ile inşa edilmiş, minimalist ve duyarlı kullanıcı arayüzü.
- **🔍 Gelişmiş Filtreleme:** Ürünleri kategorilere, popülerliğe ve fiyata göre anlık filtreleme.
- **🛒 Dinamik Sepet Deneyimi:** Akıcı ve kullanıcıyı yormayan alışveriş süreci.

## 🚀 Teknik Stack

- **Framework:** [Next.js 15](https://nextjs.org/)
- **Frontend library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management:** React Hooks & URL State
- **Icons:** [Heroicons](https://heroicons.com/)

## 🛠️ Yerel Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/Emretetik0/nextjs-commerce.git
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Geliştirme ortamını başlatın:
   ```bash
   npm run dev
   ```
